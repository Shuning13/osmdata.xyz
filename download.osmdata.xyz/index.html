Global openstreetmap data extracts | osmdata.xyz

Introduction: This project provides global data extracts based on OpenStreetMap data as GeoPackages. Each extract represents its related primary feature regarding the OpenStreetMap project.

Information about this project and data processing: osmdata.xyz on GitHub

Updates are available every month. Building and Highway are updated every odd month.

Unpacking the 7zip-archives with 7zip (GNU LGPL license), which is available for Windows and Linux: https://www.7-zip.org/download.html.
