Global openstreetmap data extracts | osmdata.xyz

Introduction: This project provides global data extracts based on openstreetmap data as GeoPackage. Each extract represents its related primary feature regarding the openstreetmap project.

Information about this project and data processing: osmdata.xyz on github

Updates are available every month. Building and Higway are updated every second month.
